# Heritage-Treasures-An-In-Depth-Analysis-of-UNESCO-World-Heritage-Sites

Project Name:
Heritage Treasures: An in-depth analysis of UNESCO World Heritage Sites using Tableau
Overview :
This project presents an in-depth analysis of UNESCO World Heritage Sites using interactive Tableau Public dashboards integrated into a Flask-based web application. The application embeds Tableau dashboards within a responsive web interface using HTML iframes rendered through the Flask framework. It provides comprehensive insights into the global distribution of heritage sites, their categories, regional spread, and endangered status. The Flask application acts as the backend server, enabling smooth web integration and local hosting of the dashboards for interactive exploration. This project demonstrates the effective use of Flask for web integration combined with Tableau for data visualization, delivering a user-friendly analytical platform for exploring UNESCO World Heritage data.
