# Heritage-Treasures-An-In-Depth-Analysis-of-UNESCO-World-Heritage-Sites
# Project Name
Heritage Treasures – An In-Depth Analysis of UNESCO World Heritage Sites Using Tableau

## Overview
This project presents an interactive analysis of UNESCO World Heritage Sites using Tableau dashboards integrated into a Flask-based web application. The system enables users to visualize and explore heritage site distribution, categories, regions, and endangered sites through an easy-to-use and interactive web interface.
